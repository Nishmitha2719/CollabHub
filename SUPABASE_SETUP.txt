   📚 API FUNCTIONS
═══════════════════════════════════════════════════════════════

   USER FUNCTIONS:
   ───────────────
   getProjects()              → Get approved projects
   getProjects({ category })  → Filter by category
   getProjectById(id)         → Get project details
   createProject(data, skills)→ Create project (pending)
   saveProject(userId, projId)→ Bookmark project
   unsaveProject(userId, projId)→ Remove bookmark
   applyToProject(userId, projId)→ Apply to project
   getSavedProjects(userId)   → Get user's saved projects

   ADMIN FUNCTIONS:
   ────────────────
   getAllProjects()           → Get ALL projects (any status)
   approveProject(projectId)  → Set status = approved
   rejectProject(projectId)   → Set status = rejected
   deleteProject(projectId)   → Delete project
   isUserAdmin(userId)        → Check if user is admin

═══════════════════════════════════════════════════════════════
   🗂️ DATABASE SCHEMA
═══════════════════════════════════════════════════════════════

   profiles
   ├── id (UUID) → auth.users.id
   ├── name, email
   ├── role (user/admin) ⭐
   ├── bio, avatar_url, skills
   └── created_at, updated_at

   projects
   ├── id (UUID auto)
   ├── title, description, category
   ├── difficulty, duration, team_size
   ├── is_paid, budget, deadline
   ├── owner_id (UUID) → profiles.id
   ├── status (pending/approved/rejected) ⭐
   └── created_at, updated_at

   project_skills
   ├── id (UUID auto)
   ├── project_id → projects.id
   └── skill_name (text)

   applications
   ├── id (UUID auto)
   ├── user_id → profiles.id
   ├── project_id → projects.id
   ├── message, status
   └── created_at, updated_at

   saved_projects
   ├── id (UUID auto)
   ├── user_id → profiles.id
   ├── project_id → projects.id
   └── created_at

   🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

   ❌ "invalid input syntax for type uuid: '1'"
      → Use user.id from useAuth(), not "1"

   ❌ "relation 'profiles' does not exist"
      → Run supabase_collabhub_schema.sql in Supabase

   ❌ "Access denied" on /admin
      → UPDATE profiles SET role='admin' WHERE email='...';

   ❌ Browse shows no projects
      → Projects must have status='approved'
      → Update page to use getProjects()

   ❌ Can't create project
      → Must pass owner_id: user.id
